# Laboratorio-3---Respuesta-al-impulso-y-Respuesta-en-frecuencia
El objetivo de este laboratorio es validar los conceptos de identificación de sistemas abordados en la materia. Específicamente, se realizarán medidas experimentales para caracterizar un sistema linear e invariante en el tiempo (LTI) a través de su respuesta al impulso y de su respuesta en frecuencia.
